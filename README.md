# SERRAL Angola emergency website

Static one-page website prepared for GitHub Pages while the primary cPanel hosting is unavailable.

## Publish with GitHub Pages

1. Create a GitHub repository.
2. Push these files to the repository default branch.
3. In GitHub, open Settings -> Pages.
4. Set Source to "Deploy from a branch" and choose the default branch root.
5. Optional: add `serralangola.com` as a custom domain after DNS is ready.
